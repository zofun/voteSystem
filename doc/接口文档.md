### 用户注册
方法：POST
> /api/1.0/register

参数：
```json
{
  "username": "123455",
  "passwrod": "2222"
}
```
返回值：

注册成功
```json
{
  "code": 200,
  "msg": "注册成功"
}
```
用户名被占用：
```json
{
  "code": 400,
  "msg": "账号已被占用"
}
```
